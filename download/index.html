How to Install

1. Go to GitHub

Click the blue download button above to open the Jxo OS GitHub page.

2. Click “Code”

On GitHub, click the green Code button.

3. Download ZIP

Select Download ZIP to get the full offline version.

4. Extract

Right-click or two-finger click on the ZIP file and choose Extract.

5. Open Folder

Open the folder where Jxo OS was extracted.

6. Run it

Double-click index.html — Jxo OS will launch and auto-check for updates.
